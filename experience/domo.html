Developer platform foundations

Domo

Software Engineer · Mar 2015 – Aug 2019

Role focus

Built developer platforms behind large-scale local environments.

At Domo I moved from QA automation into delivery engineering, where I built internal platforms and tools that helped teams create, operate, and debug service environments.

What I built

I designed and built Kubernetes-backed developer environments that replaced local VM workflows. The platform managed many custom developer environments, scaling resources up and down as needed.

Developer tooling

I built an auto-updating Electron/Node.js CLI/GUI that tunneled a developer laptop into an environment, authenticated kubectl commands, exposed environment controls, and created or destroyed leased environments.

Platform services

I worked on Kubernetes configuration management, cluster monitoring, configuration validation, logging infrastructure with routing/filtering/transformation, certificate renewal, registry caching, and an Okta-compatible Kubernetes authentication bridge.

Why it mattered

The work turned fragile local-development infrastructure into a platform product: developers could get realistic environments without personally operating all the underlying machinery.
